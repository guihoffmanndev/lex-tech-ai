🚀 Overview

Lex is an AI-powered platform built for lawyers, law students, and legal teams who want to work faster and smarter.

It combines legal research, document drafting, and workflow automation into a single, intuitive interface — powered by cutting-edge AI.

Think: Cursor for law + ChatGPT + legal copilots.
